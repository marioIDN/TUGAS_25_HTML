Mengenai Semantic Elements di HTML 5

  • Article

    Ketika kita ingin membuat layout untuk menampilkan block artikel -artikel yang ada diwebsite kita, tag article bisa kita gunakan dengan Element header dan footer. karena tag article memiliki type block, artinya element ingi memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    Elemnet ini bisa kita gunakan untuk melengkapi layout utama. Misal Misal kita memiliki halaman blog maka kita bisa menambahkan aside dibagian bawah postingan untuk mejelaskan identitas penulis artikel. karena aside memiliki type block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
